# SQL for Data Analysis using PostgreSQL

Welcome to my SQL repository! 👋

This repository is a collection of PostgreSQL queries that I have written while strengthening my SQL skills for data analysis. It covers concepts from beginner to advanced level, with each query organized by topic and accompanied by clear comments and explanations.

The goal of this repository is to serve as both a learning resource and a portfolio showcasing my SQL proficiency.
